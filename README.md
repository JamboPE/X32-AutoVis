# Automatic Visionmixing for YSTV Live using X32 input levels

### Configuration
Change variables in `src/`

### Installation
1. Change directory to build: `cd build`
2. Change `build.sh` to be executeable: `chmod +x build.sh`
3. run `./build.sh`
